 Funcionalidades do projeto

- `Funcionalidade 1`: Cadastrar aluno.
- `Funcionalidade 2`: Buscar aluno.
- `Funcionalidade 3`: Alterar aluno.
- `Funcionalidade 4`: Excluir aluno.
- `Funcionalidade 5`: Sair do sistema.
